How to compile RSYNC on windows:

1. Install Cygwin.

2. Install the packages: make, (gcc,) gcc-core, gcc-g++, binutils and vim

3. cd to your rsync directory in Cygwin.

4. sh configure

5. make

6. copy rsync.exe (from rsync folder) and cygwin1.dll (from cygwin folder) to a new folder and zip it.
